Zadatak 1:
    U prethodnoj zadaci sam vec kreirao i klonirao Github repozitorij FIPU-OS-zadace- pomocu naredbe: git clone https://github.com/Franco-404/FIPU-OS-zadace-.git
    
    Buduci da je repozitorij vec kloniran, u ovom zadatku sam nastavio rad u postojecem direktoriju te sam stvorio novi direktorij OS_zadaca_2 i ovu datoteku vjezba_2.txt

    franco@DESKTOP-C2NOUEE:/mnt/c/Users/franc/Documents/Github/FIPU-OS-zadace-$ mkdir OS_zadaca_2
    franco@DESKTOP-C2NOUEE:/mnt/c/Users/franc/Documents/Github/FIPU-OS-zadace-/OS_zadaca_2$ touch vjezba_2.txt

Zadatak 3: 

    Kopiranje svih datoteka koje nisu skrivene u direktorij backup pomocu wildcard * i  zastavice -v koja prikazuje detalje
    franco@DESKTOP-C2NOUEE:/mnt/c/Users/franc/Documents/Github/FIPU-OS-zadace-/OS_zadaca_2/vjezba_2$ cp -v data/* backup/
        'data/script.js' -> 'backup/script.js'
        'data/style.css' -> 'backup/style.css'
    Kopiranje skrivene datoteke
    franco@DESKTOP-C2NOUEE:/mnt/c/Users/franc/Documents/Github/FIPU-OS-zadace-/OS_zadaca_2/vjezba_2$ cp -v data/.env backup/
        'data/.env' -> 'backup/.env'
    
    Alternativno je moguce sve datoteke (ukljucujuci skrivene) kopirati jednom naredbom cp -rv data/. backup/ gdje . oznacava sadrzaj direktorija data, bez kopiranja samog direktorija.

    Prebacivanje u direktorij data
    franco@DESKTOP-C2NOUEE:/mnt/c/Users/franc/Documents/Github/FIPU-OS-zadace-/OS_zadaca_2/vjezba_2$ cd data
